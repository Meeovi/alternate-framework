## AlternateCMS Official DAM Plugin

This official Digital Assest Management (DAM) plugin is a powerful and fully featured pluggable app that extends the primary functionality of AlternateCMS to support a wide range of media.

### Features include:

- Create and manage digital media across multiple channels
- Add Copyright, watermarks, approve or deny media, add terms and conditions, add location, and expiration date
- Integrates with AlternateCMS built in Tagging, Categories, Comments and Agreements functionality
- Manage media across projects, tickets, blogs, products, etc
- Organize and Search media in grid, mosiac, or list mode
- Each media has tons of options to provide them with alot of information including Comments, dates, tags, etc
- Share media across numerous platforms including Facebook, Twitter, Meeovi, Pixanomy, Instagram
- Extend his plugin with other plugins including the official Facebook, Pixanomy, Intagram plugins
- Add your branding to media through watermarks
- Secure your images, download them, upload them to S3 or your own server
- Other features include Visual Previews, Reporting on each Media, Create Media Dashboards, built in Video and Audio Players with support for high-definition and 4k

### How to Install

1. (In AlternateCMS) -> click Menu -> System -> Integrations -> (In Integrations Marketplace) Search and Install AlternateCMS Official DAM Plugin
2. Plugin functionality will be available within the Content Menu integrated into the Media Manager
3. Features could potentially be extended further by other plugins and themes.
