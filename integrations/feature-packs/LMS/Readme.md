## AlternateCMS Official Learning Mangement System (LMS) plugin

The official Learning Management System (LMS) plugin offers numerous functionality to build intiutive learning environments. 

### Features included.

- Create virtual schools complete with classes, students, virtual learning environments
- Create classrooms with exams, quizzes, a grading system, and video calling
- Take notes with audio clips, images, and document sharing
- Store your materials like books, previous quizzes, exams, and other documents in a virtual folder
- A digital board great for coding, taking notes, collaborating, and more
- This plugin integrates with AlternateCMS' built features including Projects, Tickets, Checklists, Rewards, Agreements, and more.

- Provide training to others with created classes, quizzes, exams, and competency or letter grading system.

### How to Install

1. (In AlternateCMS) -> click Menu -> System -> Integrations -> (In Integrations Marketplace) Search and Install AlternateCMS Official LMS Plugin
2. Plugin functionality will be available within the Content Menu
3. Features could potentially be extended further by other plugins and themes.
