## AlternateCMS Official Newsroom Plugin

This official Customer Relationship Managment (Newsroom) plugin is a powerful and fully featured pluggable app that extends the primary functionality of AlternateCMS to support a wide range of customers and news sources.

### Features include:

- Create and send emails to your customers with built in Email functionality.
- Integrates with AlternateCMS built-in Blog, Categories, Tags, Projects, Tickets, Internationalization, Products, and Pages functionalities.
- Extends the Blog Functionality using the pre-exisiting features and adding automation, paper trail with file attachments, article dark mode, built-in sharing, and commenting

- Add Fact Boxes, footnotes, jargon busters, and reading previews.
- Change font size, background colour, and line breaks
- Save articles as templates for new or customized articles
- Workflow functionality for team collaboration
- Built-in Spell and Grammar checking with official plugins
- Can be extended with plugins, apps, and themes

### How to Install

1. (In AlternateCMS) -> click Menu -> System -> Integrations -> (In Integrations Marketplace) Search and Install AlternateCMS Official Newsroom Plugin
2. Plugin functionality will be available within the Content Menu
3. Features could potentially be extended further by other plugins and themes.
