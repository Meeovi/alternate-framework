# Node demo

This demo serves a static HTML page that initializes `createMediaEngine` in the browser.

## Files

- `media-demo.html`: browser demo page
- `server.ts`: tiny Node static server

## Setup

1. Install dependencies in your app:
   - `npm i @mframework/alternate-media`
2. Build/bundle the page with your preferred toolchain, or adapt to your existing static setup.
3. Run server with:
   - `tsx demo/node/server.ts`

## Try

- Visit `http://localhost:3000/media-demo`
